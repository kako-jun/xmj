# 邪雀 Xtreme Mahjong (xmj) - プロジェクトビジョン

## プロジェクト概要

「邪雀 Xtreme Mahjong (xmj)」は、「流れ」や「オカルト」といった非合理的な要素や、各種創作麻雀の特殊ルールをテーマにした、新しい形の麻雀ゲームです。

### コアコンセプト

1. **緊張感とカオス**: 福本伸行作品に登場するような極限状況での駆け引き
2. **非合理性の導入**: 従来の合理的な麻雀戦術にはない、「オカルト」「流れ」の要素
3. **分散型アーキテクチャ**: サーバー管理コストを最小限に抑えたP2P中心の技術構成
4. **クロスプラットフォーム**: CUIとブラウザの両方で遊べる

## ターゲットユーザー

- 通常の麻雀に飽きたプレイヤー
- 福本伸行作品（『アカギ』『銀と金』『天』など）のファン
- 駆け引きや心理戦を重視するゲームプレイヤー
- 新しい技術（Nostr, WebRTC）に興味がある開発者・プレイヤー

## 実装予定のゲームモード

### 1. 誠京麻雀
- 場代、二度ヅモ、役満祝儀ルール
- 原作: 福本伸行『銀と金』

### 2. 東西戦（クリア麻雀）
- 指定された二翻役5つを先に和了したチームが勝利
- チーム戦の要素を導入

### 3. 鷲巣麻雀
- 3/4が透明なガラス牌
- 血液（またはポイント）を賭けるルール
- 原作: 福本伸行『アカギ』

### 4. 闇麻
- 点棒を支払い、打牌を隠す「闇牌」
- それを看破する「照射」などのルール
- 情報戦の要素を強化

### 5. リアルタイム麻雀
- 全プレイヤーが同時に思考・打牌を行う
- 鳴きやロンは早い者勝ち（ロン > ポン > チーの優先順位）
- 制限時間内に打牌されない場合は自動でツモ切り
- アクション性の高い新しい麻雀体験

### 6. 通常麻雀（開発用）
- 標準的なリーチ麻雀ルール
- 基本機能のテストとデバッグ用

## 技術的ゴール

1. **完全なコード共通化**: RustコアをWASMにコンパイルし、CUI/Web両方で同一ロジックを使用
2. **サーバーレスP2P**: NostrとWebRTCを組み合わせた分散型オンライン対戦
3. **オフライン動作**: インターネット接続なしでシングルプレイヤーモードが完全動作
4. **匿名性**: アカウント登録不要で即時プレイ可能
5. **透明性**: オープンソースで不正のない公正なゲーム環境
